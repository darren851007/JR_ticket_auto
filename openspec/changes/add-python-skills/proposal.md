# add-python-skills

## What
Add five Python-focused Claude Code skills to `~/.claude/skills/`: `python-automation`, `python-fastapi`, `python-llm-integration`, `python-data-science`, and `python-testing`. Each skill is a folder containing a single `SKILL.md` with expert guidance tailored to the JR ticket automation project.

## Why
The current skill library only covers Android/Kotlin. The JR ticket automation project (`JR_ticket_auto`) uses Python with Playwright for browser automation, and may integrate FastAPI for status monitoring, Claude API for intelligent decision-making, pandas for availability analysis, and pytest for testing. These skills give Claude Code domain-specific best practices for each area on demand.

## Mode: Simple
Simple: no new tests required; any existing tests must still pass.
