# select-train

## What
After the search results page loads, automatically find and select the target train by name (e.g., "Toki 307"). If the train name is not found, fall back to selecting the train with the departure time closest to a configured target time. Then select the configured seat class and click the Select button.

## Why
Phase 2 ended after submitting the search form. This phase automates the train selection step so the booking flow can proceed without manual intervention when the sale opens.

## Mode: Simple
Simple: no new tests; existing tests must still pass.
